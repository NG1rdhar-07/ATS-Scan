# ATS-Scan

Overview
AI-powered resume evaluation system that analyzes resumes for ATS optimization and provides improvement suggestions.

Tech Stack

Backend: Python, Flask
AI/ML: OpenAI API, NLP libraries
Frontend: React.js
Database: PostgreSQL

Features

Resume parsing and analysis
ATS keyword optimization
Score calculation
Improvement recommendations
Industry-specific suggestions

Status
🚧 Currently in development
